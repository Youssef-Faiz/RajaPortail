# RajaPortail
Titre du Projet : Portail Intelligent pour les Joueurs du Raja Casablanca
Description :Application mobile permettant aux joueurs du Raja de gérer leurs profils, consulter leurs statistiques, emplois du temps et notifications. Le staff peut suivre les performances, planifier les entraînements et communiquer avec les joueurs. Une IA analyse les données pour fournir des recommandations personnalisées.
Technologies Suggérées :Android Studio (Java), Spring Boot, MySQL, Power BI, Python (IA)
Thèmes Principaux :Gestion des joueurs, suivi de performance, communication interne
